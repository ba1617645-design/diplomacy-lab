# Diplomacy Lab 🎯

منصة تعلم تفاعلية للدبلوماسية المجتمعية - Interactive Learning Platform

## 🚀 الميزات
- 5 ورش تدريبية تفاعلية
- محاكاة التفاوض مع 6 فرق
- محرك ذكاء اصطناعي للتحليل والتقارير
- لوحة تحكم للمدرب
- نظام مزامنة عبر Supabase
- Gamification (XP، شارات، مستويات)

## 🛠️ التقنيات
- React 19 + TypeScript
- Vite + Tailwind CSS
- Supabase
- Netlify

## 📦 التشغيل
```bash
npm install
npm run dev
```

## 🌐 النشر
```bash
npm run build
# ارفع مجلد dist/ على Netlify
```
